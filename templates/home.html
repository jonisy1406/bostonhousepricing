About This Project

The Boston House Prediction project is a classic machine learning project that aims to predict the house price (median value of owner owner-occupied homes) in Boston. The data used in this project called the Boston Housing Dataset, which contains 13 different features that are believed to be relevant in determining the price of a house. The data set can be access in sklearn library from sklearn.datasets import load_boston.

The goal of the project is to use machine learning algorithms to build a model that can accurately predict the price of a house given its features. This can be used by real estate agent, homeowners, and potential buyers to estimate the value of property.

To accomplish this, the project typically involves several steps, including data cleaning, data exploratory, features transformation, and model training adn evaluation using some algorithms. Data cleaning involves dealing with outliers, data exploration involves visualizing the data and identifiying patterns and relationship between the features and the target variable, feature transformation involves scaling using standarscaler (standarization) to make data more consistant and finally, the project involves and evaluating a machine learning model on the data.

In this project there are some algorithms that already applied such as LinearRegression, Lasso, Ridge, ElasticNet, RandomForestRegressor, DecisionTreeRegressor, and SVR. For deploy the machine learning in this web, I used model with RandomForestRegressor because this model has good perfomance compare to others algorithm. From the metrics evaluation we know that the RandomForestRegressor has more good performance than other algorithm. The values of MAE and MSE are small and R2 Score big more that other algorithm.
